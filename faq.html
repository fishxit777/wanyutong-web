// 入門使用
萬語通是什麼？怎麼開始使用？+
萬語通是一款在 LINE 群組或私訊中運作的翻譯機器人，專為台灣雇主與外籍員工日常溝通設計。

開始步驟：
  • 搜尋 LINE ID：@969wpxno，加入好友
  • 傳送任何訊息，Bot 自動偵測語言並翻譯
  • 免費版每日 15 次，無需付費即可體驗

群組版只需將萬語通加入群組，即可讓所有成員使用翻譯功能。
萬語通支援哪些語言？+
採用 DeepL API 引擎，支援 36 種語言，全部開放、無一限制。

常用語言：印尼文、越南文、泰文、菲律賓文（他加祿語）、英文、日文、韓文、簡繁中文等。

群組版 最多可同時對 8 種語言同步翻譯，是市面上同類產品支援語言最多的。
個人版和群組版有什麼差別？+
個人版：用於 LINE 私訊，適合個人對外籍友人、員工一對一溝通。

群組版：加入 LINE 群組後，群組內所有人都能享受翻譯服務，適合工廠、工地、餐廳等多人場景。群組版還包含語音翻譯、圖片OCR、忽略詞、翻譯歷史等進階功能。

// 功能說明
可以翻譯語音訊息嗎？+
可以！付費版限定

語音訊息透過 OpenAI Whisper 引擎轉錄成文字，再由 DeepL 翻譯。支援印尼文、越南文、泰文等多種語言語音輸入。群組版「體驗版」以上方案包含此功能。
可以翻譯圖片中的文字嗎？（OCR）+
可以！商務版限定

直接在 LINE 傳送圖片，萬語通會自動辨識圖片中的文字（OCR）並翻譯。適合用於合約、公文、藥品說明書等書面文件。採用 Claude AI 進行圖片文字辨識，準確率高。
什麼是「翻譯確認」功能？+
翻譯確認是萬語通獨有的功能。翻譯完成後，外籍員工可以對翻譯結果按 👍 / 👎 給予回饋，讓雇主知道對方是否真正理解了訊息內容。這解決了「翻譯出來但對方看不懂」的常見問題。
什麼是「忽略詞」功能？+
忽略詞讓您設定不需要翻譯的特定字詞，例如人名、地名、公司內部專有名詞。設定後，包含該詞的訊息不會觸發翻譯，避免誤翻或群組訊息被不必要地打斷。
什麼是「勿擾模式」？+
勿擾模式可讓群組在指定時段（例如午休、下班後）暫停翻譯功能，避免 Bot 在非工作時間持續回應干擾群組成員。
// 安全與信任
翻譯引擎是哪家？準確嗎？+
文字翻譯使用 DeepL API，在多項翻譯品質評比中獲得最高評分，尤其亞洲語言（印尼文、越南文）表現優於 Google 翻譯與一般 AI 翻譯。

語音翻譯使用 OpenAI Whisper（語音轉文字）＋ DeepL。圖片 OCR 使用 Claude AI。
我的對話內容會被保存或外洩嗎？+
翻譯歷史僅保存 最近 100 筆，供用戶自行查詢。資料儲存於加密的 PostgreSQL 資料庫，不會對外公開，也不會用於廣告或第三方分析。伺服器設有 API rate limiting 與爬蟲防護機制。
Bot 突然無法使用怎麼辦？+
萬語通部署在 Render.com Starter 方案（不休眠），並由 UptimeRobot 每 5 分鐘監控一次，異常時立即處理。

若遇到無法使用的情況，請聯繫客服 fishxit，我們會優先排查。付款後 24 小時內 Bot 完全無法回應者，可申請退費。

# 官網 FAQ 方案付款連結同步紀錄

日期：2026-06-26

## 本次調整

- FAQ「方案有哪些？」已同步為：官網方案按鈕與 LINE 官方帳號付款入口會直接引導使用者到對應付款流程。
- FAQ「如何升級或延長方案？」已同步為：不再要求使用者先選擇或輸入細分方案指令，改由方案按鈕或官方帳號入口完成。
- 中英文切換使用的 FAQ 字串也同步更新，避免切換語言後又出現舊流程。

## 對應 Bot 行為

- LINE Bot 後端已改為付款入口集中管理，月費版、半年版、一年版、尊爵版可由官網方案按鈕或官方帳號入口取得付款連結。
- 付款連結點擊後才建立訂單，避免查詢方案時產生多筆未付款訂單。

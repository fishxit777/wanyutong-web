# 萬語通加入開通流程影片嵌入紀錄

更新日期：2026-06-26

## 目的

- 取代原本較抽象的加入開通影片。
- 新版影片改用實際官網畫面與 LINE QR 畫面，照真實操作順序呈現。
- 讓新用戶不用讀長文，也能看懂：選方案、點加入、進入 LINE、掃碼成功。

## 新版影片流程

1. 收費方案頁：先看到所有付費方案與加入開通按鈕。
2. 游標移到「加入開通」按鈕，按鈕高亮。
3. 點擊「加入開通」，出現點擊波紋。
4. 轉場到官方 LINE 加入頁。
5. 游標移到「手機直接加入 LINE」按鈕，按鈕高亮。
6. 點擊「手機直接加入 LINE」。
7. 轉場到 LINE QR Code 頁。
8. QR Code 高亮，提示掃碼加入好友。
9. 顯示「掃碼成功」與勾選提示。
10. 結尾整理 4 步驟：選方案、加入官方 LINE、放進群組、開始翻譯。

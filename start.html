四步驟開始使用Start in Four Steps

先加入萬語通 LINE 官方帳號，再放進群組，設定語言，開始傳訊息。付款與開通疑問請聯絡客服協助。Add the WanyuTong LINE account, place it in your group, set languages, and start messaging. Contact support for payment or activation questions.

最短導入路徑Shortest Setup Path

不用建立新 App、不用搬群組。把 Bot 加進既有 LINE 群組，就能從文字翻譯開始。No new app and no group migration. Add the bot to the existing LINE group and start with text translation.

1. 加入萬語通1. Add WanyuTong

掃 QR Code 或點手機直接加入 LINE，先成為官方帳號好友。Scan the QR code or tap the mobile LINE button to add the official account.

2. 放進群組2. Add to Group

把 Bot 邀請進工作群，讓主管與員工在原本群組裡使用。Invite the bot into the work group so everyone stays in the original LINE group.

3. 設定語言3. Set Language

用 @語言 查看設定，依群組成員切換印尼、越南、泰文等語言。Use language commands to check and switch Indonesian, Vietnamese, Thai, and more.

4. 開始傳訊息4. Start Messaging

文字、圖片、語音都可進入翻譯流程；付費版可再用摘要與報表。Text, images, and voice can enter the translation flow; paid plans add summaries and reports.

AI 祕書指令AI Secretary Commands

@祕書 說明、@祕書 狀態、@祕書 摘要、@祕書 報表，照順序使用最清楚。Use Help, Status, Summary, and Report commands in order for the clearest workflow.
